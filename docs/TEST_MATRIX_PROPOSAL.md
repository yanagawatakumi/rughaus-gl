# テストマトリクス提案（v0.1）

最終更新日: 2026-04-09
対象: RUGHAUS GL（Shopify Advanced / LPO + B2B BSS）

## 1. 前提
- 価格正本: `通常価格 + LPO追加料金（隠し商品行）` を基準に、B2B割引を最後に適用
- B2B初期条件: 顧客タグ判定（国条件なし）
- 言語/通貨: 自動判定 + 手動上書き
- 対応言語: EN/DE/FR/IT/ES/ZH-TW/KO
- 対応通貨: USD/EUR/KRW/TWD/HKD

## 2. テスト軸
- 顧客種別: 一般 / B2Bタグ
- 商品種別: Stock / Custom（LPO）
- 言語選択: 自動 / 手動
- 通貨選択: 自動 / 手動
- 市場: US / HK / IT
- 導線: PDP -> Cart -> Checkout（ドラフト経由含む）

- P-01: `商品表示通常価格 + LPO追加料金` がCart小計と一致

- P-03: B2B非対象では割引が適用されない
- P-04: 顧客タグ付け/解除で割引挙動が切り替わる
